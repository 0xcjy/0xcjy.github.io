示例

格式化前

for (int i = 1; i <= n; i ++) 
{
    // 输出结果
    cout << "Hello, world! i = " << i << endl;
    
    /* 输出格式：
       i: 索引值
    */
    if (i % 5 == 0) {
        cout << "Multiple of 5: " << i << endl;
    }
}

格式化后

for(int i=1;i<=n;i++){
    cout<<"Hello, world! i = "<<i<<endl;
    if(i%5==0){
        cout<<"Multiple of 5: "<<i<<endl;
    }
}

格式化功能

  • 删除所有 字母与非下划线符号之间 的空格
  • 删除所有 非下划线符号之间 的空格
  • 删除所有 ) 与 { 之间 的所有空白（包括换行、空格、制表符）
  • 删除所有 // 与换行符之间 的内容（单行注释）
  • 删除所有 /* 与 */ 之间 的内容（多行注释）
  • 删除所有 空行
  • 保留代码原有的 换行和缩进 结构
  • 保护 字符串内容（引号内的所有内容不受影响）
  • 正确处理 转义引号（如 \" 不会被识别为字符串结束）
