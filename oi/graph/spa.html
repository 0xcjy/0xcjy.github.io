最短路算法流程

1. Floyd（O(n3)O(n^3)O(n3)，多源最短路径）

  1. 初始化一个 n×nn \times nn×n 的距离矩阵 dist\text{dist}dist，其中 disti,j\text{dist}_{i,j}disti,j​ 表示顶点 iii 到顶点 jjj 的初始距离：
    • 若 i=ji = ji=j，则 disti,j=0\text{dist}_{i,j} = 0disti,j​=0；
    • 若顶点 iii 和 jjj 之间有直接边，disti,j\text{dist}_{i,j}disti,j​ 设为边的权重；
    • 否则，disti,j=∞\text{dist}_{i,j} = \inftydisti,j​=∞（无穷大）。
  2. 遍历所有中间顶点 kkk（从 111 到 nnn）：
    • 遍历所有起点 iii（从 111 到 nnn）：
      • 遍历所有终点 jjj（从 111 到 nnn）：
        • 如果 disti,j>disti,k+distk,j\text{dist}_{i,j} > \text{dist}_{i,k} + \text{dist}_{k,j}disti,j​>disti,k​+distk,j​，则更新 disti,j=disti,k+distk,j\text{dist}_{i,j} = \text{dist}_{i,k} + \text{dist}_{k,j}disti,j​=disti,k​+distk,j​。
  3. 算法结束后，disti,j\text{dist}_{i,j}disti,j​ 即为顶点 iii 到顶点 jjj 的最短路径距离。

2. 朴素 Dijkstra（O(n2)O(n^2)O(n2)，无负权边）

  1. 给定起点 sss，初始化距离数组 dist\text{dist}dist：dists=0\text{dist}_s = 0dists​=0，其他顶点 disti=∞\text{dist}_i = \inftydisti​=∞；初始化一个标记数组 visited\text{visited}visited，所有元素为 false\text{false}false（表示未访问）。
  2. 执行 nnn 次循环（nnn 为顶点数）：
    • 在未访问的顶点中，找到距离起点最近的顶点 uuu（即 distu\text{dist}_udistu​ 最小且 visitedu=false\text{visited}_u = \text{false}visitedu​=false）。
    • 标记 uuu 为已访问（visitedu=true\text{visited}_u = \text{true}visitedu​=true）。
    • 遍历 uuu 的所有邻接顶点 vvv：
      • 如果 distv>distu+边(u,v)的权重\text{dist}_v > \text{dist}_u + \text{边}(u, v)\text{的权重}distv​>distu​+边(u,v)的权重，则更新 distv=distu+边(u,v)的权重\text{dist}_v = \text{dist}_u + \text{边}(u, v)\text{的权重}distv​=distu​+边(u,v)的权重。
  3. 算法结束后，disti\text{dist}_idisti​ 即为起点 sss 到顶点 iii 的最短路径距离。

3. 堆优化 Dijkstra（O(mlog⁡n)O(m\log n)O(mlogn)，无负权边）

  1. 给定起点 sss，初始化距离数组 dist\text{dist}dist：dists=0\text{dist}_s = 0dists​=0，其他顶点 disti=∞\text{dist}_i = \inftydisti​=∞；使用一个优先队列（小根堆），存储 (距离,顶点)(\text{距离}, \text{顶点})(距离,顶点) 对，初始将 (0,s)(0, s)(0,s) 入堆。
  2. 当 优先队列不为空 时：
    • 弹出堆顶元素 (current_dist,u)(\text{current\_dist}, u)(current_dist,u)（即当前距离起点最近的顶点 uuu）。
    • 如果 current_dist>distu\text{current\_dist} > \text{dist}_ucurrent_dist>distu​，则跳过该顶点（已处理过更优路径）。
    • 遍历 uuu 的所有邻接顶点 vvv：
      • 计算新距离 new_dist=distu+边(u,v)的权重\text{new\_dist} = \text{dist}_u + \text{边}(u, v)\text{的权重}new_dist=distu​+边(u,v)的权重。
      • 如果 new_dist<distv\text{new\_dist} < \text{dist}_vnew_dist<distv​：
        • 更新 distv=new_dist\text{dist}_v = \text{new\_dist}distv​=new_dist。
        • 将 (new_dist,v)(\text{new\_dist}, v)(new_dist,v) 入堆。
  3. 算法结束后，disti\text{dist}_idisti​ 即为起点 sss 到顶点 iii 的最短路径距离。

4. Bellman-Ford（O(nm)O(nm)O(nm)，能处理负权边）

  1. 给定起点 sss，初始化距离数组 dist\text{dist}dist：dists=0\text{dist}_s = 0dists​=0，其他顶点 disti=∞\text{dist}_i = \inftydisti​=∞。
  2. 执行 n−1n-1n−1 次循环（nnn 为顶点数）：
    • 遍历所有边 (u,v)(u, v)(u,v)：
      • 如果 distu≠∞\text{dist}_u \neq \inftydistu​=∞ 且 distv>distu+边(u,v)的权重\text{dist}_v > \text{dist}_u + \text{边}(u, v)\text{的权重}distv​>distu​+边(u,v)的权重，则更新 distv=distu+边(u,v)的权重\text{dist}_v = \text{dist}_u + \text{边}(u, v)\text{的权重}distv​=distu​+边(u,v)的权重。
  3. 检测负权环：
    • 再次遍历所有边 (u,v)(u, v)(u,v)：
      • 如果 distu≠∞\text{dist}_u \neq \inftydistu​=∞ 且 distv>distu+边(u,v)的权重\text{dist}_v > \text{dist}_u + \text{边}(u, v)\text{的权重}distv​>distu​+边(u,v)的权重，则说明图中存在负权环，无法计算最短路径。
  4. 若不存在负权环，disti\text{dist}_idisti​ 即为起点 sss 到顶点 iii 的最短路径距离。

5. SPFA（O(km)∼O(nm)O(km)\sim O(nm)O(km)∼O(nm)，能处理负权边）

  1. 给定起点 sss，初始化距离数组 dist\text{dist}dist：dists=0\text{dist}_s = 0dists​=0，其他顶点 disti=∞\text{dist}_i = \inftydisti​=∞；使用一个队列存储待松弛的顶点，初始将 sss 入队；初始化一个入队次数数组 cnt\text{cnt}cnt，cnts=1\text{cnt}_s = 1cnts​=1，其他为 0。
  2. 当 队列不为空 时：
    • 出队顶点 uuu。
    • 遍历 uuu 的所有邻接顶点 vvv：
      • 计算新距离 new_dist=distu+边(u,v)的权重\text{new\_dist} = \text{dist}_u + \text{边}(u, v)\text{的权重}new_dist=distu​+边(u,v)的权重。
      • 如果 new_dist<distv\text{new\_dist} < \text{dist}_vnew_dist<distv​：
        • 更新 distv=new_dist\text{dist}_v = \text{new\_dist}distv​=new_dist。
        • 如果 vvv 不在队列中：
          • 将 vvv 入队。
          • cntv+=1\text{cnt}_v += 1cntv​+=1。
          • 如果 cntv≥n\text{cnt}_v \geq ncntv​≥n（nnn 为顶点数），则说明存在负权环，算法终止。
  3. 若未检测到负权环，disti\text{dist}_idisti​ 即为起点 sss 到顶点 iii 的最短路径距离。
